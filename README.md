# eslint-config

An unopinionated ESLint config.

## Installation

Install using Yarn.

```bash
yarn add --dev @allanchau/eslint-config
```

## Requirements

- Node.js v10.

## Usage

Create an eslint file.

```json
{
  "extends": ["@allanchau/eslint-config"]
}
```
